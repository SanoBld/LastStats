Picks the right label for the current localeNotifier value, falling back to French. Central helper so any data table (stat cards, accent names…) can support all 5 languages without per-call ternaries.

Implementation

String pickLang({
  required String fr,
  required String en,
  required String es,
  required String zh,
  required String pt,
}) {
  switch (localeNotifier.value) {
    case 'en': return en;
    case 'es': return es;
    case 'zh': return zh;
    case 'pt': return pt;
    default:   return fr;
  }
}
