# Icon

Place your icon here as **icon.png** (this exact name and path: `assets/icon.png`).

- Square image, 512×512 px recommended (a smaller square like 192×192 also works).
- PNG with a transparent or solid background.

It's already wired up in `index.html`, `download.html`, and `recap.html` —
once the file is here, it shows up automatically as the browser tab favicon
and as the icon if someone adds the site to their phone's home screen.
No other change needed.
