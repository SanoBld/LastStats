How do I get an API key?
  1. Log in at last.fm
  2. Go to last.fm/api/account/create
  3. Fill in Application name = "LastStats"
  4. Click Submit
  5. Copy the API key (32 chars)
100% free. Scrobbling must be active on your profile.

What is a One-Hit Wonder?

In LastStats, a One-Hit Wonder is an artist you've listened to very few times — between 1 and 3 scrobbles. These are musical curiosities that never became habits. The list below shows your top 20.
