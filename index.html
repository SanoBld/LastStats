Comment obtenir une clé API ?
  1. Connectez-vous sur last.fm
  2. Allez sur last.fm/api/account/create
  3. Remplissez Application name = "LastStats"
  4. Cliquez Submit
  5. Copiez l'API key (32 chars)

Qu'est-ce qu'un One-Hit Wonder ?

Dans LastStats, un One-Hit Wonder est un artiste que vous avez écouté très peu de fois — entre 1 et 3 scrobbles. Ce sont les curiosités musicales qui ne sont jamais devenues des habitudes. La liste ci-dessous vous montre vos top 20.
