Comment obtenir une clé API ?
  1. Connectez-vous sur last.fm
  2. Allez sur last.fm/api/account/create
  3. Remplissez Application name = "LastStats"
  4. Cliquez Submit
  5. Copiez l'API key (32 chars)
Création 100% gratuite. Le scrobbling doit être actif sur votre profil.
